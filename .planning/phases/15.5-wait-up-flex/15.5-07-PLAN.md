Plan 07 — Cache bump + cross-repo deploy ritual.

Final wave-3 closeout. Bumps `sw.js` CACHE so installed PWAs invalidate on next online activation, then runs the cross-repo deploy ritual per RUNBOOK §H. Deploy ordering is critical:

1. **queuenight functions FIRST** (`firebase deploy --only functions` from `~/queuenight/`) — installs the Plan 05 reaction-push Branch 2 + reactionPosted in NOTIFICATION_DEFAULTS. 2. **couch hosting SECOND** (`bash scripts/deploy.sh 35.5-wait-up-flex` from couch repo) — ships the UI changes (Plans 01-04 + 06) + bumped sw.js CACHE.
3. **UAT** in order: Scripts A + B + E (single device) → Scripts C + D (two devices or CF emulator).
