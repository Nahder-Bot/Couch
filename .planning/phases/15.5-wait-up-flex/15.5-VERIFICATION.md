# Phase 15.5: Wait Up Flex Verification Report

**Phase Goal:** Convert the existing fixed-cap reaction-delay mechanism into a flexible "Wait Up" feature that supports any delay from 0 to multi-hour, in both sport-mode and movie-mode watchparties, with a chip + nudge + custom-picker UX. Strip reaction content from server-side push templates when the receiver has Wait Up active so spoiler text stops landing on lock screens before the moment arrives.
