<objective>
Surface watchparty entry/joining in the primary Tonight flow so it's not buried behind the ⋯ action sheet. PARTY-01: Tonight's Pick card gains a "Watch together" CTA. PARTY-07: Tonight screen shows a banner whenever a party is live or imminent (<30 min out), with Join. Reuses the existing `openWatchpartyStart` / `openWatchpartyLive` functions — no new modals.

Purpose: Discoverability. Currently the only way to start a watchparty is via ⋯ on a title card, which users don't think to tap during the main spin-reveal flow. Banner solves the reverse discoverability — you arrive in the app, someone else has a party live, you should know.
Output: Button added to spin-result modal; banner container on Tonight screen; render function + dismiss logic.
</objective>
