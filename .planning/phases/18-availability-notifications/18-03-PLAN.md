Plan 03 — Smoke gate for the pure helpers exported in Plan 18-01.

Phase 18 introduces three pure functions in queuenight/functions/index.js:
- `addedBrandsFor(oldProviders, newProviders)` — D-07 subscription-bucket diff
- `buildPushBody(matches)` — D-14 single + D-15 batch push body builder
- `normalizeProviderName(raw)` — TMDB → canonical brand collapse (mirrors couch-side)

These fns own the user-visible push-body voice and the cross-repo D-07 diff
contract — regressions ship straight to production push notifications. They're
pure (no Firestore / no network / no browser) so a Node-only smoke test catches
regressions in <100ms with zero infrastructure.

This plan creates `scripts/smoke-availability.cjs` (mirrors the existing
`scripts/smoke-position-transform.cjs` + `scripts/smoke-tonight-matches.cjs`
pattern from Phase 15.5/15.5.5) and wires it into `scripts/deploy.sh` §2.5
(Pure-function smoke gate) so every deploy validates the contracts before
firebase publishes.

**Why a separate plan rather than bundle into 18-01 or 18-02:** 18-01 writes to
queuenight (cross-repo); 18-02 writes to couch js/app.js. This plan writes to
couch scripts/ — non-overlapping with both, can run truly parallel in Wave 1.
Bundling smoke into 18-01 would force the smoke into the queuenight repo (which
deploy.sh doesn't run) or duplicate the helper definitions (DRY violation).
A dedicated couch-side smoke that requires() the queuenight module is the
cleanest split.

**Why graceful skip when queuenight is absent:** Per CLAUDE.md + RUNBOOK §H,
queuenight is a sibling repo not always present on every dev machine. If a fresh
clone of couch ships without queuenight nearby, deploy.sh shouldn't hard-fail on
the smoke. The smoke detects the missing module and prints SKIP + exit 0.
Production deploys (where queuenight IS present) catch real regressions.

rules-emulator coverage is moot (no new rules); CF unit test for `addedBrands`
computation + smoke test for push body builder"):**

    let fails = 0;

    function eq(label, got, want) {
      const ok = got === want;
      if (!ok) {
        console.error('FAIL ' + label + ': got ' + JSON.stringify(got) + ' want ' + JSON.stringify(want));
        fails++;
      } else {
        console.log('OK   ' + label);
      }
    }

    function eqSet(label, gotSet, wantArr) {
      // Compare a Set against an expected array (order-independent).
      if (!(gotSet instanceof Set)) {
        console.error('FAIL ' + label + ': got non-Set ' + JSON.stringify(gotSet));
        fails++;
        return;
      }
      const want = new Set(wantArr);
      if (gotSet.size !== want.size) {
        console.error('FAIL ' + label + ': size mismatch got ' + gotSet.size + ' want ' + want.size);
        fails++;
        return;

      for (const b of want) {
        if (!gotSet.has(b)) {
          console.error('FAIL ' + label + ': missing brand ' + b);
          fails++;
          return;
        }
      }
      console.log('OK   ' + label);
    }

    function eqObj(label, got, want) {
      const a = JSON.stringify(got);
      const b = JSON.stringify(want);
      if (a !== b) {
        console.error('FAIL ' + label + ':\n  got:  ' + a + '\n  want: ' + b);
        fails++;
      } else {
        console.log('OK   ' + label);
      }
